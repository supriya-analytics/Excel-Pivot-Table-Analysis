# Sales Analysis Report Using Excel Pivot Tables

## Project Overview
This project analyzes sales data using Excel Pivot Tables, Pivot Charts, and Slicers to understand
category-wise, region-wise, and segment-wise performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dataset
- Sample E-commerce Sales Dataset
- Columns: Order Date, Category, Region, Segment, Sales, Profit, Profit Margin

## Key Insights
- Sports category generates the highest sales.
- South region leads total sales.
- Consumer segment dominates sales.
- West region has the highest profit margin.
- Electronics and Clothing are the most profitable categories.

## Files
- Sales_Dashboard.xlsx – Interactive dashboard
- Sales_Analysis_Report.pdf – Business insights
- Sales_Data.xlsx – Raw data
